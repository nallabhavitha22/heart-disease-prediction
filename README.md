# Heart Disease Prediction System

The Heart Disease Prediction System is a machine learning–based application designed to predict whether a person is likely to have heart disease based on various medical attributes. The system analyzes patient health data and uses predictive algorithms to assist in the early detection of heart-related conditions.

## Features

- Predicts the likelihood of heart disease using patient health parameters
- Machine learning model built using LightGBM
- RESTful API developed using Flask for real-time predictions
- User input interface for entering medical attributes
- Efficient data preprocessing and feature engineering

## Tech Stack

- **Programming Language:** Python  
- **Machine Learning:** LightGBM, Scikit-learn  
- **Backend API:** Flask  
- **Data Processing:** Pandas, NumPy  

## Model Performance

- Achieved **92% prediction accuracy** through feature engineering and hyperparameter tuning.

## How It Works

1. The user provides medical attributes such as age, blood pressure, cholesterol levels, etc.
2. The data is processed and passed to the trained machine learning model.
3. The model predicts whether the patient is likely to have heart disease.
4. The prediction result is returned through the Flask API.
